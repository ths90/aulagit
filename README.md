# Aula de Git
Aula de utilização de git <br>
e comando básicos.

#Comandos Básicos
git clone: Comando para clonar projetos <br>
git status: Comando para dar várias informações sobre um determinado projeto <br>
git add: Comando para preparar o arquivo para versionamento <br>
git commit -m: modifica a mensagem do commit mais recente / comentários das alterações <br>
git push: Comando para publicar <br>
git pull: Comando para atualização de projeto <br>
